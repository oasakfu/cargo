Thank you!

Thank you for playing Cargo, the epic snail adventure game!

Installation

Cargo does not need to be installed; simply double click on the game file to play.
  • GNU+Linux: Double-click on cargo/cargo
  • Windows: Double-click on cargo/cargo.exe
  • Mac OSX: Double-click on cargo.app

Playing the game

Cargo can crawl on almost any surface, including walls and ceilings. Use the arrow keys to move. To turn a corner, press the left or right arrow and forward or backward - as if you were driving a car*.

Cargo's shell is useful, too! Press Space to enter the shell. You can then roll along the ground by pressing the arrow keys. The shell won't stick to walls or the ceiling, but it can float. Are there any other uses for it?

* You know, with arrow keys. That's how I drive my car.

Keyboard and mouse controls

KeyEffect
W or Up-arrowMove forward
S or Down-arrowMove backward
A or Left-arrowTurn left
D or Right-arrowTurn right
Move mouseMove camera (look around)
Space, Left-mouse or Enter/ReturnEnter shell, accept dialogue option
X, Right-mouseDrop shell, cancel dialogue option
C, Middle-mouseReset camera position
Q, Mouse-wheel-upNext shell, next option
E, Mouse-wheel-downPrevious shell, previous option
EscapeShow quit options
F10Quit immediately

Joystick/joypad controls

These default joystick bindings are set up for a Playstation 3 controller. If your controller is different, you may need to change them from the main menu.
KeyEffect
JoypadMove relative to snail (like WSAD keys)
Joystick 1Move relative to camera
Joystick 2Move camera (look around)
Cross (X)Enter shell, accept dialogue option
Circle (O)Drop shell, cancel dialogue option
TriangleReset camera position
L1Next shell, next option
R1Previous shell, previous option
StartShow quit options

Cargo is free software

Free software doesn't mean "free of charge" - it means that this game respects your freedom. It does this by endorsing your rights to use the game in any way you like, study the source code, modify the game and share it with your friends. These are fundamental values of the free software movement; a movement for the freedom of people who use software.

Unlike many games, Cargo is also free of DRM and other technical restrictions.
