Installation

Unzip the .zip or .tar file appropriate for your system. Use the 32-bit version if you have a 32-bit CPU or operating system; otherwise, use the 64-bit version.

  • GNU/Linux: 32 or 64bit?
  • How about Ubuntu?
  • Windows: 32 or 64bit?
  • Only recent Mac computers are supported, so only the 64-bit version is available for OS X.

After unzipping, refer to readme.html in the game folder for more information.

Known Issues (bugs!)

This is a beta release for testing purposes. The Windows version should work well, but there are known issues on Mac OS X and GNU/Linux.

  • On some GNU/Linux systems, the first level will fail to start.
  • On Mac computers with "integrated" or Intel video cards, the frame rate of animations is low (even if the overall frame rate is high). This happens even if a "discrete" graphics card (such as nVidia) is present too.
